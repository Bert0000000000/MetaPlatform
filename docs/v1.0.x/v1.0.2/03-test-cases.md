# v1.0.2 端到端测试用例

> **版本**：v1.0.2「Link+Export」
> **配套文档**：[`01-user-stories.md`](./01-user-stories.md) · [`04-test-report-template.md`](./04-test-report-template.md) · [`07-migration-from-v1.0.1.md`](./07-migration-from-v1.0.1.md)
> **目的**：列出 v1.0.2 全部必须通过的测试用例，作为自动化 + 手工测试的依据。

---

## 〇、用例编号规范

```
TC-{Epic}-{编号}

例：TC-L1-001 = L1 Epic 的第 1 个用例
    TC-NF-001 = 非功能（性能/安全）的第 1 个用例
    TC-INT-001 = 集成的第 1 个用例
    TC-BC-001 = 向后兼容（v1.0.1 → v1.0.2）的第 1 个用例
```

**类型**：
- **E2E**（End-to-End）：跨前后端，使用 Playwright
- **Integration**：使用 supertest 跨服务调用
- **Unit**：单文件级别（Jest/RTL）
- **Security**：渗透 + 越权 + 注入
- **Performance**：压测（k6 或 JMeter）
- **Compat**：向后兼容测试
