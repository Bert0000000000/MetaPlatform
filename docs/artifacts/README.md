# 项目产物归档

本目录用于集中存放非源码类产物，避免散落在仓库根目录。

## 目录说明

| 目录 | 用途 |
|---|---|
| `verification/screenshots/` | 功能验证、UI 回归截图 |
| `verification/logs/` | 前后端 dev server、调试日志 |
| `deliverables/` | 设计交付物、迁移路线图、Canvas 原型等 |

## 注意

- `verification/screenshots/` 与 `verification/logs/` 已加入 `.gitignore`，不会进入版本控制。
- 需要保留的正式设计文档、路线图请放在 `deliverables/` 或 `docs/` 下对应分类目录。
