数据流说明

数据流 来源 → 目标 技术 说明
CDC 实时入湖 ERP/CRM → Hudi Debezium + Kafka + Flink + Hudi 数据库变更实时捕获，写入 Hudi MoR 表，支持 Upsert/Delete
批量 ETL 外部系统 → Hudi/Iceberg Airflow + DBT 定时批量同步，DBT 做 ODS→DWD→DWS→ADS 分层建模
湖仓联邦查询 Hudi/Iceberg → StarRocks StarRocks External Catalog StarRocks 直接查询 Hudi/Iceberg 表，无需数据搬运
数据映射建模 Hudi/PG → Ontology TECH-ONT 映射引擎 将数据表字段映射到 Ontology Entity/Attribute，建立语义层
知识入库 文档 → Milvus + Neo4j TECH-RAG + Embedding 文档解析→分块→Embedding→Milvus 向量索引；实体关系→Neo4j 图谱
AI 问答检索 用户 → Milvus + Neo4j → LLM TECH-RAG + TECH-LLMGW 混合检索（向量+BM25+图谱）→ Reranker → LLM 生成答案
Action 执行 Ontology → 外部系统 TECH-ACTION + API 规则触发或手动触发 Action，通过 API 写回外部系统
MCP 工具暴露 平台 → 外部 AI 工具 TECH-MCP (MCP Server) Ontology 查询、RAG 检索、Action 执行能力暴露给 Cursor/Copilot
A2A 任务委托 平台 ↔ 外部 Agent TECH-A2A (A2A Protocol) 数字员工与外部 Agent 互相发现能力、委托任务、异步返回结果
