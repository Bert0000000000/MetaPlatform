# TECH-RULE - 规则引擎服务

## 模块类型

TECH 模块

## 作用

业务规则引擎，支持 Drools/自研规则表达式，为工作流、Action、Agent 提供规则计算能力。

## 上游依赖

- `TECH-ONT`

## 下游消费

- `TECH-WFE`
- `TECH-ACTION`
