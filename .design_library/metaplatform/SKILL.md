# MetaPlatform Design System

## Overview
MetaPlatform 设计系统 — 本体论驱动的企业级 AI 平台视觉规范。

## Design Tokens
- Token CSS: `colors_and_type.css`
- Token JSON: `css.json`

## Components (6)
- `button` — 按钮 (Primary/Ghost)
- `card` — 卡片 (Default/Stat)
- `table` — 表格
- `input` — 输入框 (Default/Search)
- `badge` — 状态标签 (5 语义色)
- `sidebar` — 侧边栏导航

## Key Design Decisions
- Compact spacing (4px base), small radius (3-6px)
- Border-only cards (no shadows)
- Dark navy sidebar (#0f172a), white canvas
- Primary: #3b82f6 (blue-500)
- Font: Inter + Noto Sans SC
