# TECH-A2A - A2A 协议适配服务

## 模块类型

TECH 模块

## 作用

Agent-to-Agent Protocol 适配层，支持 Mate 数字员工与外部 Agent 的能力发现、任务委托、异步协作。

## 上游依赖

- `TECH-AGENT`
- `TECH-WFE`

## 下游消费

- `外部 A2A Agent`
